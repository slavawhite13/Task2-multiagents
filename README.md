Практическое задание номер 2 по курсу "Мультиагентные технологии". Выполнил Вековцев Вячеслав Владиславович.

Принцип работы алгоритма по этапам. 
Этап 1: Агенты знают своё число и связи с другими агентами. 
Этап 2: Агенты отправляют своё число остальным доступным агентам. 
Этап 3: Агенты реализуют протокол локального голосования и, используя одну ячейку памяти, обновляют своё значение на каждом этапе. 
Этап 4: этапы 2 и 3 повтооряются до тех пор, пока количество итераций алгоритма не достигнет заданного значения.
При передаче данных также добавлены помехи -0.5 - +0.5 и потеря связей.
В консоль выводятся значения агентов через каждые три этапа. Также для проверки удалённых рёбер выводится информация и о них.

Алгоритм запуска программы (для среды разработки IntelliJ IDEA Community Edition):
1. Скопирывать ссылку на гитхаб из письма.
2. В гитхабе скачать файлы jade.jar и main.java.
3. Открыть среду разработки IntelliJ IDEA. Выбрать File, далее New..., далее Project from version control. Вставить ссылку на гитхаб и нажать clone.
3. Открыть File, Project Structure, Modules, Dependencies, нажать на плюс и добавить JARs or directories. В открывшемся окне указать путь к jade.jar. Нажать ОК.
